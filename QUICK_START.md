# دليل البدء السريع - Quick Start Guide

## للعملاء 👥

### 1. عرض العائلات
```
اذهب إلى: https://digitmine.com/shop
```

ستجد:
- 🏠 واجهة جديدة تعرض العائلات كبطاقات
- 🔍 شريط بحث للبحث عن عائلة أو منتج
- 🔄 أزرار للتبديل بين "العائلات" و "المنتجات"

### 2. البحث عن عائلة
```
أدخل اسم الخدمة في شريط البحث:
- Netflix
- Spotify
- Adobe
- ChatGPT
- إلخ...
```

### 3. عرض العروض
```
اضغط على بطاقة العائلة لرؤية:
- جميع المنتجات المتاحة
- الأسعار المختلفة
- المدد المختلفة
- التقييمات والمبيعات
```

### 4. الشراء
```
اضغط على منتج:
1. اختر نوع الحساب (خاص/مشترك/عائلي)
2. اختر المدة (شهر/3 أشهر/سنة/إلخ)
3. أضف إلى السلة
4. أكمل الشراء
```

## للمسؤول 👨‍💼

### 1. الوصول إلى لوحة التحكم
```
اذهب إلى: https://digitmine.com/admin/families
```

ستجد:
- 📊 جميع العائلات والمنتجات والعروض
- 🔍 شريط بحث للبحث عن عائلة
- ➕ زر لإضافة منتج جديد

### 2. إدارة العائلة
```
اضغط على عائلة لرؤية:
- جميع المنتجات تحتها
- جميع العروض المتاحة
- معلومات المخزون
- الأسعار
```

### 3. تعديل العرض
```
اضغط على أيقونة التعديل (✏️) بجانب العرض:
1. عدّل المدة
2. عدّل السعر بالدينار
3. عدّل السعر بالدولار
4. عدّل المخزون
5. عدّل المورد
6. اضغط "حفظ"
```

### 4. إظهار/إخفاء العرض
```
اضغط على أيقونة العين (👁️):
- العروض المخفية لا تظهر للعملاء
- يمكنك إظهارها لاحقاً
```

### 5. حذف العرض
```
اضغط على أيقونة الحذف (🗑️):
1. تأكيد الحذف
2. سيتم حذف العرض نهائياً
```

## استيراد البيانات 📥

### الخطوة 1: تحضير الملف
```
تأكد من أن لديك ملف CSV يحتوي على:
- العائلة
- اسم المنتج
- المدة
- السعر
- نوع الحساب
- نوع العرض
```

### الخطوة 2: التشغيل
```bash
# في مجلد المشروع
npx ts-node scripts/import-csv-data.ts
```

### الخطوة 3: التحقق
```
اذهب إلى /admin/families
يجب أن ترى جميع العائلات والمنتجات
```

## الأسئلة الشائعة ❓

### س: كيف أضيف عرضاً جديداً؟
**ج:** 
1. اذهب إلى `/admin/families`
2. اضغط على العائلة
3. اضغط على المنتج
4. اضغط على "تعديل"
5. اضغط على "عرض جديد"

### س: كيف أغير السعر؟
**ج:**
1. اذهب إلى `/admin/families`
2. اضغط على العائلة
3. اضغط على أيقونة التعديل بجانب العرض
4. عدّل السعر
5. اضغط "حفظ"

### س: كيف أخفي عرضاً؟
**ج:**
1. اذهب إلى `/admin/families`
2. اضغط على العائلة
3. اضغط على أيقونة العين بجانب العرض
4. سيتم إخفاء العرض

### س: هل يمكن العودة إلى عرض المنتجات الفردية؟
**ج:** نعم! اضغط على زر "المنتجات" في الأعلى

### س: كيف أبحث عن عائلة معينة؟
**ج:** استخدم شريط البحث وأدخل اسم العائلة أو المنتج
